# Search Wiki Prompt

Use this prompt template to search the wiki effectively.

## Basic Search

```
Search the wiki for: [your topic here]

Example: Search the wiki for: python asyncio
```

## Filtered Search

```
Search the wiki for: [query]
Domain: [domain-name]
Kind: [page|entity|concept|source]
Tags: [tag1, tag2]

Example:
Search the wiki for: kubernetes
Domain: homelab
Kind: concept
Tags: k8s, container
```

## Common Search Patterns

### Find a Specific Technology
```
Search the wiki for: [technology name]
Kind: entity
```

### Find Concepts and Ideas
```
Search the wiki for: [concept name]
Kind: concept
```

### Find All Pages in a Domain
```
Search the wiki for: *
Domain: [domain-name]
```

### Find Pages by Tag
```
Search the wiki for: *
Tags: [tag-name]
```

## Tips

- Use specific queries for better results
- Combine domain and tag filters for precision
- Check multiple domains if unsure where content lives
